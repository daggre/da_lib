# da_lib Documentation

Welcome to the documentation for da_lib, a library of shared functions and utilities for RedM resources.

- [Installation](getting-started.md#installation)

## Features

da_lib includes the following feature modules:

### Cache System
- [Lazy Cache](features/cache/lazy.md) - Caching with customizable time-to-live
- [Delay Cache](features/cache/delay.md) - Delayed execution cache
- [Temp Cache](features/cache/temp.md) - Temporary caching

### Core Features
- [Animation](features/anim/index.md) - Animation handling
- [Audio](features/audio/index.md) - Audio management
- [Control](features/control/index.md) - User input control
- [Epoch](features/epoch/index.md) - Time management
- [Lock](features/lock/index.md) - Resource locking mechanisms
- [Mode](features/mode/index.md) - Mode switching
- [Net](features/net/index.md) - Networking utilities

### Utility Features
- [CLI](features/cli/index.md) - Command line interface tools
- [Draw](features/draw/index.md) - Drawing utilities
- [FX](features/fx/index.md) - Effects handling
- [KVP](features/kvp/index.md) - Key-value pair storage
- [Object](features/object/index.md) - Object management
- [PolyZone](features/polyzone/index.md) - Zone management
